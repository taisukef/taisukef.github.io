## プロフィール(200文字)

小学校3年時にプログラミングにはまり、福井高専卒業後、3社目の創業、株式会社jig.jpにてjigブラウザを開発。鯖江市にオープンデータを提案し、日本初のオープンデータ都市誕生、毎日何か創って発表する「一日一創」を始める。「すべてのこどもたちにプログラミングを」PCN共同創始者で、こどもパソコンIchigoJam開発者。子供から大人まで幅広く、楽しいプログラミングとデータ活用術を伝えている。
